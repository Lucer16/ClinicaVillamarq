**Villamarq** es una plataforma web moderna y de alto rendimiento diseñada para una clínica veterinaria líder en Penonomé. El sitio prioriza la experiencia del usuario (UX) y el SEO avanzado para conectar a los dueños de mascotas con servicios médicos de última generación.

---

## 🚀 Características Principales

* **Diseño Premium:** Interfaz ultra-moderna utilizando Glassmorphism y animaciones fluidas.
* **SEO Avanzado:** Implementación de datos estructurados (JSON-LD), Open Graph y meta-etiquetas optimizadas para Google.
* **Experiencia Single-Page:** Sistema de modales dinámicos para servicios, evitando tiempos de carga innecesarios.
* **Responsive Design:** Optimización total para dispositivos móviles y tablets.
* **Integración Directa:** Botones de agendamiento rápido vía WhatsApp y mapas interactivos.

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura semántica y diseño moderno.
* **Tailwind CSS:** Framework de utilidades para un estilo rápido y escalable.
* **JavaScript (ES6+):** Lógica para el manejo de modales, sliders y filtrado de datos.
* **Font Awesome 6:** Iconografía profesional.
* **Google Fonts:** Tipografía "Inter" para máxima legibilidad.
